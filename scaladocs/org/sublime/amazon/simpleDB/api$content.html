SimpleDB API for Scala 0.9-SNAPSHOT API

This document is the API specification for SimpleDB API for Scala 0.9-SNAPSHOT API

Class Summary
trait SimpleAPI extends Concrete
This trait provides a simple API for accessing Amazon SimpleDB. It is designed to be mixed in to objects or classes that provide the necessary networking capabilities.
class SimpleDBAccount (val awsAccessKeyId : java.lang.String, awsSecretKey : java.lang.String) extends SimpleAPI
A simple concrete implementation of the simpleDB API. Instantiate one of these with your AWS Credentials and then use the methods defined in the SimpleAPI trait.
