class Item(val domain : Domain, val name : java.lang.String)
extends AnyRef
A class which serves as a proxy to an Item within simpleDB. This class holds none of the attributes of the item itself. Calls to methods which read or write attributes to and from the item will result in network requests to SimpleDB.

Update the contents of this item with a map of attributes names and a set of values and boolean for each one. If the boolean is true, the existing values will be replace with those in the set, otherwise the set of values will be added to the existing ones. This is the analog of the 'PutAttributes' request.
